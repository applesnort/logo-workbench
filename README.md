# logo-workbench

Two self-contained HTML tools for building wordmarks — pick a font, stack the words, tune kerning, tracking, weight, and case. No build step, no dependencies.

- **`logo-workbench.html`** — grid of font candidates for a given phrase; lock the ones you like, tweak per-card controls.
- **`logo-explore.html`** — companion explorer for deeper variations.

## Usage

Open either file directly in a browser:

```sh
open logo-workbench.html
```
